Addendum:

The Lights Out game required numerous important phases to create. I started by concentrating on comprehending the principles and regulations of the game, which enabled me to create the game board using an HTML grid style. The main features of the game were modeled by each square in the grid, which had a light that could be turned on or off.

Implementing the logic to switch nearby squares when a player clicked on a square was one of the biggest obstacles I had to overcome. This meant paying close attention to the grid's design and the relationships between each square and its neighbors. I was able to write a JavaScript code that successfully toggled nearby squares through iterative testing and debugging.

I made the decision to include a hint button in order to improve the gameplay. This button gives players useful hints for resolving the puzzle, giving the game a more strategic aspect. In order to make the game more visually appealing and to provide players a more engaging experience, I also added a backdrop image.

I gained an understanding of the value of user input and interactivity in game design throughout this project. I was able to enhance the user experience and improve gameplay by iteratively testing the game and getting input from other players. Overall, working on this project has improved my grasp of CSS and JavaScript, and I'm excited to use these newfound abilities in other projects.

Instructions

The Lights Out game is played on a grid of squares, where each square represents a light. The goal of the game is to turn off all the lights.

Clicking on a square toggles the state of that square and its adjacent squares (up, down, left, right). If a light is off, clicking on it turns it on, and vice versa.

The game starts with a random configuration of on and off lights. To win the game, you need to turn off all the lights in the grid.

Use the "Reset" button to start a new game and the "Hint" button to get a helpful hint to solve the puzzle.
